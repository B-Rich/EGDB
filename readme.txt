#==============================================================================+
#   Epic Gamers Item Database built from the myfirstplugin by nlstart
#
# 	This is an e107 plugin for keeping track of item drops in the Earth &
#	Beyond emulator project.
#
#==============================================================================+

Prerequisites:
==============
* E107 core v0.7.7 (or newer) installed.
* Preferably a proper developer editor installed (e.g. [link=http://notepad-plus.sourceforge.net/]Notepad++[/link])
